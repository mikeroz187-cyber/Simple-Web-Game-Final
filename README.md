# Studio Empire

An adult content production management simulator. Build a studio from amateur beginnings to industry dominance.

## Project Status

**Current Phase:** MVP (Act 1 only)

## Repository Structure

- `/.codex/` — Codex AI configuration
- `/docs/mvp/` — MVP implementation docs (follow these)
- `/docs/vision/` — Full game design (reference only, do not implement)
- `/src/` — Game source code

## Documentation

Start here:
1. `docs/mvp/CURRENT-SCOPE.md` — What to build now
2. `docs/mvp/BUILD-ORDER.md` — Step-by-step implementation guide
3. `docs/mvp/TECHNICAL-GUARDRAILS.md` — Technical constraints

For full game vision (post-MVP reference), see `docs/vision/` folder.

## Quick Start

Open `src/index.html` in a browser. No build tools required.
