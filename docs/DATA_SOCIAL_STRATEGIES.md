# Data — Social Strategies (Current)

This catalog documents the social strategy options and manual strategy settings used by `src/systems/social.js` and `src/config.js`.

## Strategy Catalog (Fixed in Code)

| id | label | description | primaryEffect | instagramReachMult | xReachMult | subscriberConversionMult |
| --- | --- | --- | --- | --- | --- | --- |
| balanced | Balanced | Keep reach steady across platforms while maintaining conversion stability. | No modifiers to platform reach or subscriber conversion. | 1.0 | 1.0 | 1.0 |
| growth_focus | Growth Focus | Push for follower growth, even if conversion efficiency dips. | Instagram reach x1.2, X reach x0.9, subscriber conversion x0.9. | 1.2 | 0.9 | 0.9 |
| revenue_focus | MRR Focus | Prioritize higher‑value audiences over raw reach. | Instagram reach x0.9, X reach x1.2, social sub conversion x1.1. | 0.9 | 1.2 | 1.1 |

## Manual Social Strategy (Config)

| key | value | meaning |
| --- | --- | --- |
| channels | ["tease", "collabs", "ads"] | Allocation channels shown in the UI. |
| channelLabels | {tease:"Tease", collabs:"Collabs", ads:"Ads"} | Display labels for channels. |
| minSpend | 0 | Minimum daily budget. |
| maxSpend | playerCash | Maximum daily budget (player cash). |
| defaultDailyBudget | 200 | Default budget on new saves. |
| followersPerDollar | tease:1.2, collabs:1.6, ads:0.8 | Base followers gained per dollar. |
| subsPerFollower | 0.01 | Social subs per follower generated by manual strategy. |
| diminishingReturnsK | 0.002 | Diminishing‑returns factor for large spends. |

## Config Mapping
Values map to:
- Strategy catalog: `src/systems/social.js` (`SOCIAL_STRATEGY_CATALOG`).
- Default strategy: `CONFIG.social.strategy.defaultStrategyId`.
- Manual strategy: `CONFIG.social.manualStrategy`.
